Website-- Life Tracker
This website helps track important goals , tasks like health, studies , habits and current status of each which are pending. We can add tasks to be completed with the title and time at which they should be completed which will help us to keep track of time and be more disciplined towards life therefore the name Life Tracker.
HTML, JavaScript , CSS were used to build the website.
